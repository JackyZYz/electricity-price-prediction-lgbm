# 基于 WT-LightGBM 的用户侧日前出清电价预测系统使用手册

## 1. 项目简介

本项目实现了一个基于**小波变换 + LightGBM（WT-LightGBM）**的电力现货市场日前出清电价预测系统。核心思路是使用**平稳小波变换（SWT）**将电价序列分解为长度相等的低频/高频分量，对每个分量分别训练 LightGBM 模型，预测时通过逆平稳小波变换（`iswt`）重构得到最终的 96 点日前电价。

主要能力：

- 读取 `Dataset/` 目录下的多源 CSV 数据（15 分钟粒度）
- 自动构建时序特征、构造特征与日期特征
- 训练 WT-LightGBM 模型，并生成评估报告
- 预测指定日期的 96 点电价

## 4. 模型训练

### 4.1 默认训练

使用默认配置文件 `config/default.yaml` 训练 WT-LightGBM 模型：

```bash
python -m src.cli.main --config config/default.yaml train
```

运行流程：

1. 读取 `Dataset/` 下的目标变量与各类特征表
2. 校验数据质量（缺失、异常、时间连续性等）
3. 构建特征矩阵并保存到 `data/features/`
4. 训练 WT-LightGBM 模型
5. 在测试集上评估并生成报告
6. 保存模型到 `models/wtlgbm.pkl`

### 4.3 切换为纯 LightGBM（可选）

编辑 `config/default.yaml`：

```yaml
model:
  type: "lgbm"    # 将 wtlgbm 改为 lgbm
```

重新执行训练命令即可。

### 4.4 常用训练参数（`config/default.yaml`）

```yaml
model:
  type: "wtlgbm"                # 模型类型
  decompose_level: 2            # SWT 分解层数
  lgbm_params:
    learning_rate: 0.05         # 学习率
    num_leaves: 63              # 叶子数
    max_depth: 12               # 树深度
    num_boost_round: 5000       # 最大迭代轮数
    early_stopping_rounds: 50   # 早停轮数
```

## 7. 常见问题

**Q1：训练时报 `pyarrow` 未安装？**

```bash
pip install pyarrow>=12.0.0
```

**Q2：预测时报特征维度不一致？**

通常是因为 `config/default.yaml` 中的特征配置与训练模型时不一致，或模型文件是旧版本。解决：重新执行 `train` 生成新的模型。

**Q3：数据集路径与默认配置不一致？**

修改 `config/default.yaml` 中的 `data.dataset_root` 和 `data.sources` 对应路径即可。

## 8. 配置文件关键项

详见 `config/default.yaml`，主要模块：

- `data`：数据路径与源文件映射
- `preprocessing`：缺失值、异常值、归一化策略
- `features`：滞后窗口、滚动窗口、日期特征开关
- `model`：模型类型与 LightGBM 超参数
- `output`：模型、预测、报告、特征存储路径
